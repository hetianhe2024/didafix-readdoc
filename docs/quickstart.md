## 📖 功能使用教程

### 1. 图片转PDF

将多张图片合并为一个PDF文件。

**操作步骤：**

1. **上传图片**
   - 点击"图片转PDF"工具
   - 点击上传区域或拖拽图片文件
   - 支持JPG、PNG、BMP、WebP等格式

2. **调整顺序**
   - 拖拽图片缩略图调整顺序
   - 点击删除按钮移除不需要的图片

3. **设置参数**
   - 选择PDF页面大小（A4、A5、Letter等）
   - 选择页面方向（横向/纵向）
   - 设置边距（可选）

4. **转换下载**
   - 点击"开始转换"按钮
   - 等待处理完成
   - 点击"下载"保存PDF文件

!!! example "使用示例"
    **场景：**将手机拍摄的合同照片转为PDF
    
    1. 上传5张合同照片
    2. 按页码顺序排列
    3. 选择A4纸张、纵向
    4. 转换并下载PDF

### 2. 图片格式转换

快速转换图片格式，如JPG转PNG、PNG转WebP等。

**操作步骤：**

1. **上传图片**
   - 点击"图片格式转换"工具
   - 上传需要转换的图片

2. **选择目标格式**
   - 在下拉菜单中选择目标格式
   - 可选：JPG、PNG、WebP、BMP、GIF

3. **开始转换**
   - 点击"开始转换"
   - 等待处理完成
   - 下载转换后的文件

**格式选择建议：**

| 使用场景 | 推荐格式 | 原因 |
|---------|---------|------|
| 网页显示 | WebP | 体积小，加载快 |
| 需要透明背景 | PNG | 支持透明通道 |
| 照片存储 | JPG | 压缩率高，通用性好 |
| 动图 | GIF | 广泛支持 |

### 3. AI智能抠图

自动识别主体，去除背景，生成透明背景图片。

**操作步骤：**

1. **上传图片**
   - 点击"AI智能抠图"工具
   - 上传需要抠图的图片

2. **AI自动处理**
   - 系统自动识别主体
   - 分离前景和背景
   - 生成预览图

3. **查看效果**
   - 查看抠图效果
   - 可选择不同背景颜色预览

4. **下载结果**
   - 点击"下载"保存PNG格式（透明背景）
   - 可批量处理多张图片

!!! tip "抠图技巧"
    - 主体边缘清晰的图片效果最好
    - 避免主体与背景颜色过于接近
    - 人像、商品类图片识别效果最佳

**应用示例：**

=== "证件照"
    **场景：**制作白底证件照
    
    1. 上传个人照片
    2. AI自动抠图
    3. 选择白色背景预览
    4. 下载白底证件照

=== "商品图"
    **场景：**电商商品白底图
    
    1. 上传商品照片
    2. AI去除背景
    3. 选择白色背景或透明背景
    4. 下载用于电商平台

=== "头像制作"
    **场景：**制作透明背景头像
    
    1. 上传人像照片
    2. AI抠出人物
    3. 保持透明背景
    4. 下载PNG格式

### 4. OCR文字识别

识别图片中的文字，转为可编辑文本。

**操作步骤：**

1. **上传图片**
   - 点击"OCR文字识别"工具

3. **查看和编辑**
   - 查看识别的文字内容

4. **导出结果**

**应用场景：**

### 6. 图片圆角处理

为图片添加圆角效果，让图片更美观。

**操作步骤：**

1. **上传图片**
   - 点击"图片圆角处理"工具
   - 上传需要添加圆角的图片

2. **调整圆角**
   - 拖动滑块调整圆角半径（0-50%）
   - 实时预览效果

3. **选择背景**
   - 透明背景（PNG格式）
   - 或选择纯色背景

4. **下载图片**
   - 点击"下载"保存处理后的图片

**圆角半径参考：**

- **5-10%** - 轻微圆角，适合卡片、按钮
- **15-25%** - 明显圆角，适合头像框
- **50%** - 圆形效果，适合头像

### 7. 图片压缩

压缩图片文件大小，加快加载速度，节省空间。

**操作步骤：**

1. **上传图片**
   - 点击"图片压缩"工具
   - 上传需要压缩的图片（支持批量）

2. **选择压缩质量**
   - **高质量**：90-95%，轻微压缩
   - **标准**：75-85%，平衡压缩
   - **高压缩**：60-70%，大幅减小

3. **查看对比**
   - 查看压缩前后对比
   - 显示文件大小和压缩率

4. **下载图片**
   - 单张下载或批量下载为ZIP

!!! info "压缩建议"
    - 网页图片：建议压缩到200KB以内
    - 社交分享：建议500KB以内
    - 照片存储：使用高质量模式

### 8. 图片加水印

为图片添加文字或Logo水印，保护版权。

**操作步骤：**

1. **上传图片**
   - 点击"图片加水印"工具
   - 上传需要加水印的图片

2. **选择水印类型**
   
   **文字水印：**
   - 输入水印文字
   - 选择字体、大小、颜色
   - 调整透明度

   **图片水印：**
   - 上传Logo图片
   - 调整大小和透明度

3. **调整位置**
   - 拖动水印到合适位置
   - 或选择预设位置（左上、右上、居中等）

4. **批量添加**（可选）
   - 上传多张图片
   - 统一添加相同水印

5. **下载图片**
   - 点击"下载"保存带水印的图片

**水印位置建议：**

- **右下角** - 常见位置，不影响主体
- **平铺** - 防止裁剪盗图
- **居中** - 强调版权声明

## ❓ 常见问题

### 文件上传问题

**Q: 支持哪些图片格式？**

A: 支持常见的所有图片格式：JPG、JPEG、PNG、WebP、BMP、GIF、TIFF等。

**Q: 单个文件大小限制是多少？**

A: 
- 免费用户：单个文件最大10MB
- 会员用户：单个文件最大50MB

**Q: 一次最多可以上传多少张图片？**

A: 
- 免费用户：最多10张
- 会员用户：最多100张

### 处理速度问题

**Q: 处理速度受什么影响？**

A: 主要受以下因素影响：
- 图片大小和数量
- 网络速度
- 服务器负载
- 会员用户享有更快处理速度

**Q: 处理失败怎么办？**

A: 
1. 检查网络连接
2. 确认图片格式正确
3. 尝试减小图片大小
4. 刷新页面重试

### 隐私安全问题

**Q: 上传的图片会被保存吗？**

A: 
- 所有图片在浏览器本地处理
- 处理完成后自动删除
- 不会上传到服务器（部分AI功能除外）
- 详见 [隐私政策](about.md)

**Q: 数据安全吗？**

A: 
- 使用HTTPS加密传输
- 临时文件定期清理
- 不会泄露给第三方
